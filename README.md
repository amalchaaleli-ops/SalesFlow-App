## 🎯 À propos du projet

**SalesFlow Analytics** est une application Web interactive (SaaS) développée en Python. Elle permet aux entreprises de e-commerce d'automatiser l'analyse de leurs fichiers de ventes massifs, dépassant ainsi les limites des tableurs classiques.

## 🚀 Fonctionnalités Clés

- **Smart Upload :** Chargement dynamique de fichiers CSV. L'application vérifie automatiquement l'intégrité des données entrantes.
- **Moteur de Calcul (Pandas) :** Calcul instantané du Chiffre d'Affaires Brut, du CA Net (après remises) et de la TVA (20%).
- **Dashboard Automatisé :** Affichage des KPI majeurs (CA Total, TVA Totale, Produit Star).
- **Data Visualization (Plotly) :** Génération de graphiques interactifs pour identifier rapidement les tendances de rentabilité par produit.

## 🛠️ Stack Technique

- **Frontend / Interface Web :** Streamlit
- **Data Engineering & Analyse :** Pandas
- **Data Visualization :** Plotly Express
- **Déploiement Cloud :** Streamlit Community Cloud
